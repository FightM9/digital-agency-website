## How to use

1. Clone or  download from GitHub
2. Install Node Modules: **npm i**
3. Run: **gulp**

## Gulp tasks:

```
gulp build
```

Optimization CSS, JS, HTML, IMG, SVG. create WebP, SVG sprite. Create build folder and copy source files, live server + watcher
